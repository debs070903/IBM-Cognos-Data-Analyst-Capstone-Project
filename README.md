# IBM Cognos Data Analyst Capstone Project

This repository contains the capstone project completed as part of the **IBM Data Analyst Professional Certificate**.  
The project demonstrates an end-to-end data analytics workflow, including data extraction, cleaning, transformation, analysis, and visualization using **IBM Cognos Analytics** and Python-based tools.

---

## ğŸ¯ Project Objective

The primary objective of this project is to analyze technology and job market data to identify trends related to:
- Programming languages
- Databases
- Platforms and tools
- Compensation patterns
- Workforce demographics

The analysis aims to support **data-driven decision-making** through structured dashboards and visual insights.

## ğŸ§© Project Workflow

1. **Data Collection**
   - Data sourced through web scraping and structured datasets
   - CSV files used as primary data sources

2. **Data Cleaning & Preparation**
   - Handling missing values
   - Removing duplicates
   - Normalizing compensation-related fields
   - Filtering and transforming categorical variables

3. **Data Transformation**
   - Feature selection and aggregation
   - Scaling and normalization techniques
   - Preparing datasets for visualization tools

4. **Data Analysis**
   - Trend analysis of current vs future technologies
   - Comparative analysis across regions and demographics
   - Identification of high-demand skills

5. **Data Visualization**
   - Interactive dashboards created using **IBM Cognos Analytics**
   - Bar charts, stacked bars, maps, word clouds, and KPIs
